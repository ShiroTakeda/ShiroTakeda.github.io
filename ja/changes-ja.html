  • 2008-12-07:
    今までCSSファイル内でfont-familyの設定によりフォントの種類を指定していたが、ややこしい割りにOSの種類、フォントの有無、ブラウザの種類・設定によってころころフォントが変わってしまうので、結局指定をしないことにする。まあ、フォントにこだわる人はブラウザの方で自分で設定しているでしょうし、このサイトに関しては表示に使うフォントなんてもともとたいした問題じゃないですし。

    ページ内に画像を置くのに MODxのMaxiGallaryというsnippetを利用してみた。最近よく見る lightbox と同じような機能を持っているものです。このページで使っています。
    普通に利用できましたが、一つ問題が。snippet（の利用するCSS）内で float を　left と 指定しているため、画像の後の文章が右側に回り込んでしまうという問題です。floatの指定を書き換えてやればいいかと思いましたが、やり方がよくわからず上手くいかず。結局、次のように解決。
    まず、CSSで次の指定を追加。 
          .clearfloat {
      clear: both;
      }
    そして、画像を張り込んだ後に位置に以下を追加。 
          <div class="clearfloat"></div>
    意味もないdivタグを入れるのはスマートな方法ではないですが、とりあえずこれで解決。
