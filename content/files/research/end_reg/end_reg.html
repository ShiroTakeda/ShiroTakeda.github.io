Instruction for GAMS program.

"takeda_200703_ect_gams.zip" includes GAMS codes for implementing the simulation in the above paper (the model part). To use these codes, GTAP version 5 data and GTAPinGAMS 5.3 need to be installed properly. Extract the zip archive in the GTAPinGAMS directory and read README.txt file.

If you have any question about the program, email me.
