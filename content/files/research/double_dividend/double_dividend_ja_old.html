GAMS のプログラムの説明

"gams_program.zip" には、論文内でのシミュレーションをおこなうための GAMS のプログラムとデータが 含まれています。

ファイルの説明

ファイル名 説明
dynamic_mcp.gms メインのプログラム (中身)。これは dynamic_mcp_basic.gms から読み込まれます。
dynamic_mcp_basic.gms これも GAMS のプログラム。シミュレーションをおこなうには、これを GAMS で実行します。
set_definition.gms set の定義ファイルです。
dataset.dat データが含まれるファイル。

実行方法

上の 4 つのファイルを同じフォルダに置き、dynamic_mcp_basic.gms を GAMS で実行するだけです。残 りの3つのファイルは実行時に dynamic_mcp_basic.gms に読み込まれます。

計算内容

このプログラムでは、tax swap による効用への効果 (論文の表9の結果) のみ計算されます。結果は pc_util というパラメータに出力されます。論文内の他の計算もおこなうには、若干プログラムを書き換え る必要があります。

注意点

  • プログラム内では、モデルは mcp format で記述されています。実行し、適切な結果を得るには full license の solver PATH が必要となります。少くとも私が試したかぎりでは、GAMS にデフォー ルトで付いている solver MILES では解くことができませんでした。

  • dataset.dat に含まれるデータは加工済みのものだけです。元のデータ (連関表、3EID) を加工 するための GAMS プログラムもありますので、欲しい方には差し上げます。ただし、自分でデータを 加工するには、1995年の産業連関表、および3EIDの電子ファイルを所有している必要があります。

  • プログラムの内容、およびその他の点について不明な点がありましたら、私に問い合わせていた だけ れば、わかる範囲で説明します。
