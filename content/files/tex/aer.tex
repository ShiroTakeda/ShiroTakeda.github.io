ちなみに、Reference で一番上にある
\citet{baldwin88:_market_acces_inter_compet} が \BibTeX のデータベースで
どう登録されてるかというと、
\begin{verbatim}
	
	@InCollection{baldwin88:_market_acces_inter_compet,
	  author =       {Richard E. Baldwin and Paul R. Krugman},
	  title =        {Market Access and International Competition: A
	                 Simulation Study of {16K} Random Access Memories},
	  booktitle =    {Empirical Methods for International Trade},
	  publisher =    pu-MIT,
	  year =         1988,
	  editor =       {R. Feenstra},
	  address =      ad-MIT
	}

\end{verbatim}
こんな感じになってます。このようにデータベースでは項目毎に箇条書きで登録
しておくだけです。\BibTeX はこれを自動的に Reference 部分のような複雑な
形式に加工してくれるというわけです。
